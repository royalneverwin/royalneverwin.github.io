在jupyter里写sql

Posted on 2022-06-14 Edited on 2022-06-17

写在前面

写数据库实习，这里记录一下一些常见用法



正文

  1. 先安装：pip install ipyhton-sql
  2. 导入外部库 %load_ext sql
  3. 连接sql数据库：%sql mysql/pymysql://sql用户名:密码@IP/库名
  4. 注意，%sql执行一行sql命令，%%sql执行一整个cell的sql命令
  5. drop table if exists xxx_book：如果数据库中存在xxx_book表，就把它从数据库中drop掉。
  6. SET FOREIGN_KEY_CHECKS=0;：取消外键约束，避免表与表相互之间建立起了外键约束从而导致无法删除表与修改表结构
  7. set和declare区别：
    • SET不用明确变量的类型，DECLARE则必须要
    • SET可以定义全局或者是会话的变量，DECLARE只能用在BEGIN….END….里面，相当于是局部变量
    • SET必须赋值，DECLARE不赋值默认为NULL
    • SET的变量前面要加@，在定义和使用的时候都要加上@，而DECLARE则不用
  8. replace(string, from_str, to_str)：将string中所有出现的from_str替换成to_str。
  9. locate(subStr,string)：函数返回subStr在string中出现的位置
  10. truncate table xx：清空table
